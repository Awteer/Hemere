# HPA主翼 構造-空力統合最適設計プログラム Hemere

## 概要

本ソフトウェアは、人力飛行機の主翼設計において、これまで分断されがちだった**空力設計**と**構造設計**を統合し、実用的な計算時間で大域的な最適解を探索するための設計支援プログラムである。従来の設計手法における**構造重量の不確かさ**を排除し、構造成立性を担保した上での究極の空力性能を導出することを目的としている。2026年3月頃にバージョン1.0を配布する予定である。

## 開発の背景とモチベ

### 1. CFRP構造設計の難しさ

金属材料であれば板厚を連続的に変更して剛性を調整できるが、CFRPを用いた主翼桁は、プリプレグの積層数という**離散的な変数**によって性能が決定される。この「離散性」により、設計空間は複雑な多峰性を持つだけでなく、NP困難な整数計画問題となる。
これに対する一般的アプローチであるメタヒューリスティクス解法、代表的なもので言えば、遺伝的アルゴリズムは非常に解の探索には強力だが、主翼全幅にわたる積層構成（特に集中積層）をすべて同時に最適化しようとすると、計算量が爆発し、実用的な時間内に収束しないという課題がある。

### 2. 前作『opt-ATLAS』での知見

前身となるプロジェクト『opt-ATLAS』では、全周積層のみという簡易条件下において、構造重量を考慮した循環分布の最適化に成功した。しかし、わずか9遺伝子（設計変数）の最適化でさえ数分を要し、より現実に即した「集中積層」を導入するには計算コストが高すぎることが判明した。
