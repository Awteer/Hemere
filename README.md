HPA主翼 構造-空力統合最適設計プログラム（名称未定）
概要 (Overview)
本ソフトウェアは、人力飛行機（HPA）の主翼設計において、これまで分断されがちだった「空力設計」と「構造設計」を統合し、実用的な計算時間で大域的な最適解を探索するための設計支援プログラムです。従来の設計手法における「構造重量の不確かさ」を排除し、構造成立性を担保した上での究極の空力性能（循環分布）を導出することを目的としています。

開発の背景と課題 (Background & Motivation)
1. CFRP構造設計の難しさ
金属材料であれば板厚を連続的に変更して剛性を調整できますが、CFRP（炭素繊維強化プラスチック）を用いた主翼桁は、プリプレグの積層数（プライ数）という離散的な変数によって性能が決定されます。この「離散性」により、設計空間は複雑な多峰性（Multimodal）を持ちます。これに対する一般的アプローチであるメタヒューリスティクス解法、代表的なもので言えば、遺伝的アルゴリズム（GA）は強力ですが、主翼全幅にわたる積層構成（特に集中積層）を真面目に最適化しようとすると、計算量が爆発し、実用的な時間内に収束しないという課題がありました。

2. 前作『opt-ATLAS』での知見
前身となるプロジェクト『opt-ATLAS』では、全周積層のみという簡易条件下において、構造重量を考慮した循環分布の最適化に成功しました。しかし、わずか9遺伝子（設計変数）の最適化に数分を要し、より現実に即した「集中積層（スパキャップ）」を導入するには計算コストが高すぎることが判明しました。
3. ブレイクスルー：AIサロゲートモデルの導入
「空力最適化のフェーズで、詳細な積層構成まで決める必要はない」この発想の転換に基づき、本プログラムでは構造計算プロセスを分離（デカップリング）しました。
Phase 1 (Global): 剛性 ($EI$) と直径 ($R$) から、「上手に設計された桁重量」を瞬時に推算する。
Phase 2 (Local): 決定された剛性分布を実現する詳細な積層構成を後から逆算する。
このPhase 1を実現するために、膨大な設計パターンの構造解析（集中積層・テーパー構成含む）を行い、パレート最適解のみを学習させた決定木ベースのAIサロゲートモデル（Random Forest）を実装しました。これにより、構造計算にかかるコストを実質ゼロにし、空力最適化の試行回数を飛躍的に増やすことが可能となりました。

主な機能 (Features)
超高速重量推算:AIサロゲートモデルにより、必要な曲げ剛性と桁径から、集中積層を駆使した「最軽量の桁重量」を瞬時に算出。$D/t \le 100$ などの座屈制約や、製造可能な積層ルールを考慮済み。
高度な空力最適化:
Xfoil連携: 翼型解析による正確な空力特性の取得。
TR797循環分布: パラメータ $\beta$ を用いた柔軟かつ効率的な循環分布の表現。上反角効果の考慮:（実装予定）理論モデルに基づくロール安定性等の評価。
ねじり下げ最適化:（WIP）誘導抗力と構造変形を考慮したねじり分布の最適化。

目指すゴール
「構造のことよく分からないから、とりあえず楕円循環分布でいいや」からの脱却。構造（重量）と空力（揚抗比）のトレードオフを完璧に見極め、その機体にとって真に最適な主翼を一意に決定できるツールを目指します。
